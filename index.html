Mark I - Es un Template con Responsive Web Design! (alpha)

Plantilla personal, en HTML5, con Responsive Web Design, con soporte para tipografias Coolvetica http://www.dafont.com/coolvetica.font, Museo-700 http://www.fontsquirrel.com/fonts/Museo y Open Sans http://www.google.com/webfonts/specimen/Open+Sans

NAVEGADORES: Chrome - Safari - Rockmelt - Firefox - Opera - IE 10 (Win7) // Para IE de Win8 no funciona.

Corre sobre cualquier Smartphone o tablet.
